This page provides basic instructions for anyone who wishes to access to a select dataset of 3-D high-accuracy/resolution simulation outputs from NSF-funded research on turbulence formation and development in Convectively Breaking Shoaling Internal Solitary Waves in the South China Sea, which are available via Globus. The Globus repository contains select full-resolution 3-D snapshots from simulations of three different deep-water Internal Solitary waves, read-in and visualization files and a file outlining the core structure of the repository.

Instructions:

The individual needs to sign up for an account on Globus and then request access from P.I. P. Diamessis.

Contact me via email: Pete J. Diamessis

After signing up for Globus and requesting access, anyone who wishes to access the files should identify a receiving local Globus endpoint and enter the CAC endpoint as the source when using the Globus service (or use this URL) to transfer the desired files.
